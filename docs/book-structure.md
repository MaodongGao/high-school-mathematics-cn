# 数学分册的章节顺序

本项目区分两种顺序：

- **整理顺序**由当前最适合处理的扫描件决定，不代表最终成书顺序；
- **成书顺序**原则上沿用原稿 `数学1`、`数学2`、……的先后关系，因为这些编号大体记录了当年的学习进程。

## 已知例外：`数学4` 前置

`数学4.pdf` 的开篇是集合、集合运算、映射与函数等基础内容。它是原稿保存下来之前已经学过、后来为考试集中复习的内容，因此最终成书时应放在 `数学1.pdf` 对应内容之前。这个前置是内容依赖上的例外，不应按文件编号把它留在第四位。

现阶段章节文件名前的两位数字只是仓库内的临时装配序号。等相邻来源都完成后，再做一次集中重排，避免每新增一章就反复重命名文件和标签。

## 课纲定位

旧版普通高中数学课程标准把“计数原理、排列、组合、二项式定理及其应用”列在选修 2-3 的“计数原理”模块中；当前人教版目录仍把“排列、组合与二项式定理”作为选择性必修内容。由此采用以下依赖关系：

1. 集合、映射、函数等基础语言先行；
2. 立体几何、数列等章节按原稿学习顺序进入主体；
3. `数学18.pdf` 的“计数原理与排列组合”应位于已有基础代数之后，并与后续二项式定理、概率统计相邻；
4. 原稿末尾的错排、格路、组合几何与染色属于计数方法的深化，保留在本章末尾，不提前拆散到几何章。
